'✅' : '❌'} **CERERE ${approved ? 'APROBATĂ' : 'RESPINSĂ'}**\n` + `👤 Admin: **${user.username}**\n` + `👤 Jucător: <@${r.uid}>\n` + `💰 Sumă: **${r.amount} Cr**` }) }); await db.ref('pending_donations/' + rid).remove(); notify("ACȚIUNE FINALIZATĂ!"); }
